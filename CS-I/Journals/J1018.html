Excursions: D-type Flip Flop


EX.1:

Unlike an SR Latch, the D-type Flip Flop provides a short delay before latching outputs, which is vital for more human operators who tend to make errors more often. Additionally, the D-type is easier to use because it is controlled by just one input.


EX.2:

The two inputs in the SR Latch function as "set" and "reset" inputs, each one setting and saving their own outputs. However, the D-type Flip Flop basically has one input that can be controlled. This input changes data, while the CLK signal saves that change into the final outputs.


EX.3:

The inverter makes sure that the two AND gates never both receive identical inputs, which causes the forbidden state in the SR Latch.

I learned how to create a memory-based circuit that is more advanced than a mere SR Latch: a D-type Flip Flop. The big idea of this lesson is understanding the exact purposes of inputs and outputs, as well as other components within circuits.
