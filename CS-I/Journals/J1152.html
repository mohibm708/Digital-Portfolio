1.1:

The test is executed to see if the variable fulfills the condition, and perform the loop if it still does. If it doesn't, the loop is skipped and the program ends.


1.2:

Both the loop and the conditional use jumps and conditions to see if a variable still achieves a certain thing; however, the conditional performs in one direction while the loop moves back to the previous code block after altering the variable.


1.3:

The jump after the consequent directs back to the test to continue the loop, since the consequent is what is performed if the condition is fulfilled.
